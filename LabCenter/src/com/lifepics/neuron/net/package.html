Here's a handy summary of the differences between post transactions.

Data Hook Data To HTTPClient Headers
PostTransaction none none none
PostDataTransaction getByteData / getXMLData byte[] none
RawUploadTransaction getFile stream none
CompressUploadTransaction getFile stream (compressed Base64) none
UploadTransaction getFile plus two more stream (with prefix and suffix) multipart/form-data; boundary=?????
FormDataTransaction getFormData string (from Query) application/x-www-form-urlencoded
FormDataUploadTransaction getFormData stream (with prefix) application/x-www-form-urlencoded
getFile plus one more
SOAPUploadTransaction lots stream (with prefix and suffix) Content-Type, SOAPAction
